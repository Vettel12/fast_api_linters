# 📖 API Кулинарной книги (Recipe App)

Асинхронный сервис на **FastAPI** и **SQLAlchemy 2.0** для хранения и просмотра рецептов с автоматическим счетчиком популярности.

## 🚀 Особенности
- **Асинхронность**: Используется `aiosqlite` и `AsyncSession` для высокой производительности.
- **Умная сортировка**: Рецепты в списке автоматически сортируются по количеству просмотров, а затем по времени приготовления.
- **Счетчик просмотров**: Каждое открытие детального рецепта (`GET /recipes/{id}`) автоматически увеличивает его популярность в базе.
- **Полная документация**: Все модели и поля аннотированы для удобства фронтенд-разработки.

## 🛠 Технологии
- **FastAPI** (v0.109+)
- **SQLAlchemy** (v2.0+) — асинхронный ORM
- **Pydantic v2** — валидация данных
- **SQLite** — база данных (драйвер `aiosqlite`)
- **Pytest** — автоматизированное тестирование

# 🏁 Быстрый запуск

## 1. Клонируйте проект и создайте окружение:
```bash
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
```

## 2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## 3. Запустите сервер:
```bash
uvicorn app.main:app --reload
```

# 🔌 API Эндпоинты

* GET /recipes — Получить список всех рецептов (отсортированы по популярности).
* GET /recipes/{id} — Детальная информация + инкремент просмотров.
* POST /recipes — Создать новый рецепт.
